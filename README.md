### Week Eight

Instructions
*****
* Clone the repository
* Run bundle `
* Run `bin\rake db:migrate` to set up databases
* Type `bin/rails s`
* In a browser, navigate to ```localhost:3000```
* Sign up with as many users as you like,  post, review and endorse as much as you like!
